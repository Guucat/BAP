# 修改参考文献信息接口
> `post`   `api.lyhbio.com/rest/v1/update_reference_info`

#### 注意:发送json格式内容给后端，示例如下:
``` javascript
{
    "code": 200,
    "msg": "查询成功",
    "data": [
        {
            "reference__ref_id": "1",
            "reference__title": "蝙蝠waaa",
            "reference__pubmed_id": 1,
            "reference__doi": "nadh",
            "reference__url": "www.bianfu.com"
        }
    ]
}
```
